Instruction

This game called "Find Score in the sea"


The rule of game is: You will be given a number of questions and 2 choices. You can only choose 1. If that is correct you will get 1 point; otherwise, you lose 1 point. There is also a timer on the top, when time is out, game will stop. In addition, when your score under 0, game is over too.
Notice: Click START to begin the game.
